# AI 狼人杀 · 开发规格

## 核心参数
- 人数：支持 **6人局**（2狼、1预言家、1女巫、2村民）和 **8人局**（3狼、1预言家、1女巫、3村民）
- 发言倒计时：60秒
- 翻牌特效：快速翻开 + 光晕，不同角色光效不同
  - 蓝色 = 村民（平民光）
  - 金色 = 预言家（金光）
  - 紫色 = 女巫（紫光）
  - 红色 = 狼人（红光）
- 夜间：所有人同时选择 → 统一结算
- 胜负：显示全员身份，不标注AI/玩家

## 游戏流程

### 等待阶段
- 房间创建后，玩家列表实时显示
- 玩家可以加入（最多6人）
- 房主点击"开始游戏"

### 夜晚流程（自动推进）
1. 所有人闭眼
2. 狼人睁眼 → 选择击杀目标
3. 预言家睁眼 → 选择查验目标 → 得知结果
4. 女巫睁眼 → 得知狼人击杀目标 → 选择救人/下毒
5. 所有人闭眼 → 公布结果（带特效）

### 白天流程
1. 公布昨夜死亡（特效翻牌）
2. 遗言（若有）60秒
3. 依次发言（每轮60秒/玩家）
4. 投票（30秒倒计时）
5. 票数最高者翻开身份
   - 若平票：进入 PK 投票（平票者再次发言，其他人不说话，再投）
6. 结算胜负，或进入下一天

### 胜负条件
- 狼人全部死亡 → 好人胜利
- 狼人数量 >= 好人数量 → 狼人胜利

## 角色能力
| 角色 | 夜间能力 | 备注 |
|------|---------|------|
| 村民 | 无 | 白天发言推理 |
| 狼人 | 每夜杀一人 | 可以自刀、跳神 |
| 预言家 | 每夜查验一人 | 尽量隐藏身份 |
| 女巫 | 解药+毒药各一瓶 | 解药可自救，毒药可自用 |

## 技术方案

### 后端
- Flask + Flask-SocketIO（实时通信）
- DeepSeek LLM（AI玩家决策）
- 内存存储游戏状态

### 前端
- 纯HTML5 + CSS3 + Vanilla JS（无需构建）
- Socket.IO 客户端连接后端
- Canvas/CSS动画实现翻牌特效

## 交互细节

### 翻牌特效
- 点击"翻牌"按钮 → 牌360度旋转翻开 → 光效扩散
- 死亡揭晓：中央弹出大图，带光效+音效提示
- 身份光效：翻开的瞬间从该角色对应颜色光圈从中心炸开

### 倒计时
- 发言框底部：红色倒计时条（60→0）
- 剩余10秒：变红 + 闪烁
- 归零：自动提交/跳过

### 投票特效
- 每人头像上显示得票数（数字+进度条）
- 最高票者：红色边框
- 票死揭晓：翻牌特效

### 发言气泡
- 气泡样式：聊天气泡 + 角色小图标
- AI玩家：气泡右侧 + 轻微"思考中"动画
- 玩家自己：气泡左侧 + 蓝色

### 夜间交互
- 闭眼状态：所有人画面变暗 + "请闭眼"
- 轮到某角色：其画面亮起，其他仍暗
- 选择目标：点击玩家头像 → 高亮确认 → 确认按钮
- 所有人选完 → 自动天亮
